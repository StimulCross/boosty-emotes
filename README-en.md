## Boosty Emotes

### Languages

[Russian](README.md)

A browser extension for Chromium-based browsers and Firefox that allows you to use Twitch emotes on Boosty, including third-party providers: 7TV, FrankerFaceZ, and BetterTTV.

Add the Twitch streamers whose emotes you want to use to the extension to unlock their Twitch channel emotes across their Boosty pages.

### Usage

1. Install the extension for [Chromium](https://chromewebstore.google.com/detail/boosty-emotes/pejlmncncgbgabfhakeogmhmjinpbign) or [Firefox](YOUR_FIREFOX_LINK).
2. Log in using your Twitch account.
3. Add Twitch streamers whose emotes you want to use.

**Example:**
You can see the extension in action on my Boosty page: `https://boosty.to/stimulx`. Just add the user with the following usernames:

- **Twitch:** `stimulx`
- **Boosty:** `stimulx`

### Features

**Supported Pages & Areas:**

- **Channel Page**
- **Stream Page**
- **Chat Popup Page**
- **Main Page Feed**
- **Personal Messages**
- **Post Creation & Editing**

**Emote Capabilities:**

- Emote picker with full support for third-party emotes.
- Favorite emotes system.
- `Tab` emote autocompletion.
- Colon (`:kek`) emote autocompletion.
- Support for 7TV overlays (zero-width emotes).
- Almost full support for FFZ modifiers (except the `slide` effect).
