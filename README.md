## Boosty Emotes

### Languages

[English](README-en.md)

Браузерное расширение для Chromium и Firefox, которое даёт возможность использовать смайлы Twitch на Boosty, включая смайлы сторонних провайдеров: 7TV, FrankerFaceZ и BetterTTV.

Добавьте стримера в расширение, чтобы использовать все его смайлы с Twitch'а на его странице Boosty.

### Использование

1. Установите расширение для [Chromium](https://chromewebstore.google.com/detail/boosty-emotes/pejlmncncgbgabfhakeogmhmjinpbign) или [Firefox](YOUR_FIREFOX_LINK).
2. Войдите в расширение через свой аккаунт Twitch.
3. Добавьте стримеров, чьи смайлы вы хотите использовать.

**Пример:**
Вы можете оценить работу расширения на моей странице Boosty: `https://boosty.to/stimulx`. Добавьте пользователя со следующими никами:

- **Twitch:** `stimulx`
- **Boosty:** `stimulx`

### Возможности

**Поддерживаемые страницы и разделы:**

- **Страница канала**
- **Страница стрима**
- **Отдельное окно чата (всплывающее окно)**
- **Главная страница (лента)**
- **Личные сообщения**
- **Создание и редактирование постов**

**Возможности смайлов:**

- Меню выбора смайлов с поддержкой сторонних провайдеров.
- Избранные смайлы.
- Автозаполнение смайлов при нажатии на `Tab`.
- Автозаполнение смайлов после двоеточия (`:kek`).
- 7TV оверлеи (zero-width смайлы).
- Почти полная поддержка модификаторов FFZ (кроме эффекта `slide`).
