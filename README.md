## Boosty Emotes

### Languages

[English](README-en.md)

Браузерное расширение для Chromium и Firefox, которое даёт возможность использовать смайлы Twitch на Boosty, включая смайлы сторонних провайдеров: 7TV, FrankerFaceZ и BetterTTV.

Добавь стримера в расширение, чтобы использовать все его смайлы с Twitch'а на его странице Boosty.

### Использование

1. Установи расширение для [Chromium](https://chromewebstore.google.com/detail/boosty-emotes/pejlmncncgbgabfhakeogmhmjinpbign) или [Firefox](https://addons.mozilla.org/en-US/firefox/addon/boosty-emotes/).
2. Войди в расширение через свой аккаунт Twitch.
3. Добавь стримеров, чьи смайлы ты хочешь использовать.

**Пример:**
Ты можешь оценить работу расширения на моей странице Boosty: `https://boosty.to/stimulx`. Добавь пользователя со следующими никами:

- **Twitch:** `stimulx`
- **Boosty:** `stimulx`

### Возможности

**Поддерживаемые страницы и разделы:**

- Страница канала
- Страница стрима
- Отдельное окно чата (всплывающее окно)
- Главная страница (лента)
- Личные сообщения
- Создание и редактирование постов

**Возможности смайлов:**

- Меню выбора смайлов с поддержкой сторонних провайдеров.
- Избранные смайлы.
- Автозаполнение смайлов при нажатии на `Tab`.
- Автозаполнение смайлов после двоеточия (`:kek`).
- 7TV оверлеи (zero-width смайлы).
- Почти полная поддержка модификаторов FFZ (кроме эффекта `slide`).
